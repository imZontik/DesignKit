# DesignKit

DesignKit — минималистичный набор UI-компонентов для приложений на Android Jetpack.

## Компоненты

- **Buttons**: Button, OutlineButton, TextButton, IconButton, FAB
- **Inputs**: TextField, SearchTextField, Dropdown, Checkbox, RadioGroup, Switch
- **Feedback**: Dialog, Snackbar
- **Indicators**: Badge, LoadingIndicator
- **Layout**: Card, Divider
- **Effects**: ShimmerEffect
